# A+ Dental — Website

Static website for A+ Dental (Vancouver, BC). Customer testing/staging site.

## View the site (GitHub Pages)

After pushing to GitHub:

1. Go to **Settings → Pages**
2. Under **Source**, choose **Deploy from a branch**
3. Branch: **main** (or **master**), folder: **/ (root)**
4. Save — the site will be at `https://<username>.github.io/<repo-name>/`

## Local development

Open `index.html` in a browser, or use a simple local server (e.g. `npx serve .`).

## Structure

- **HTML:** `index.html`, `about.html`, `services.html`, `team.html`, `contact.html`, `new-patients.html`
- **Styles:** `style.css`
- **Scripts:** `script.js`, `data.js`, `populate.js`
- **Assets:** `imgs/`
